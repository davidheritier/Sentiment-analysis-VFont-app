# run

## install dependencies

pip install
- Flask
- Flask-SQLAlchemy
- Flask-WTF
- textblob


## start web app

In your terminal emulator, go to the location where the files are and type:
```
python3 run.py
```

## open address

Works in Google Chrome only (MacOS)
```
localhost:5000 (depending on what the server returned)
```
